# Microbiome Twin Study Source Data

This repository contains source data and R scripts for reproducing Figures 1-5 in the microbiome twin study.

## Contents

The `source_data/` directory includes the following files:

- `fig1.R`: R script for Figure 1
- `fig1.RData`: R data file for Figure 1
- `fig2.R`: R script for Figure 2
- `fig2.RData`: R data file for Figure 2
- `fig3.R`: R script for Figure 3
- `fig3.RData`: R data file for Figure 3
- `fig4.R`: R script for Figure 4
- `fig4.RData`: R data file for Figure 4
- `fig5.R`: R script for Figure 5
- `fig5.RData`: R data file for Figure 5

## Usage

To reproduce each figure, open R or RStudio, set the working directory to the repository root, and run the corresponding R script.

For example:

```r
source("source_data/fig1.R")
```
